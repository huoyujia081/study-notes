# 质数问题

## 判断n是否是素数

方案一：从2~n-1试探是否能整除n

方案二：从2~根号n试探是否能够整除n

## 埃氏筛法

用于求多个素数，如2到100000之间的所有的素数。方便打表。

```c++
int main() {
    vector<int> num(10001);
    for(int i = 2; i <= 100000; i++) {
        if(num[i] == 0) {
            printf("%d\n", i);
        }
        for(int j = 2 * i; j <= 100000; j += i) {
            num[j] = 1;
        }
    }
}
```

## 快速幂

计算a^n^，快速幂的思路是使用递归，这样的时间复杂度为logn

1. n是偶数，a^n^ = a^n/2^ * a^n/2^
2. n是奇数，a^n^ = a^n/2^ * a^n/2^ * a

快速幂可以用于矩阵运算
